Поле противника

Доступные команды:

1. Управление игрой:
create master/slave - создать игру в режиме master/slave
start - начать игру
stop - остановить игру
save filename - сохранить игру
load filename - загрузить игру

2. Настройка поля:
set size width height - установить размер поля
set ships size count - установить количество кораблей
set strategy type - установить стратегию (ordered/random/custom)

3. Размещение кораблей:
place x y size direction - разместить корабль
  x y - координаты (0-9)
  size - размер корабля (1-4)
  direction - направление (h/v)
Примеры:
  place 0 0 4 h - 4-палубный корабль горизонтально в точке 0,0
  place 2 3 2 v - 2-палубный корабль вертикально в точке 2,3

4. Игровой процесс:
shot x y - сделать выстрел по координатам
exit - выйти из игры
